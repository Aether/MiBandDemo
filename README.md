该项目为针对小米手环BLE蓝牙协议进行操作，从而实现小米手环APP的所有功能，同时也可根据现有功能进行创意发挥，实现手机通知振动、久坐提醒等功能。
由于精力有限，项目进度不一定很快，故特地撰写教程放在博客上。欢迎大家进行二次开发，但请附带本博客及GitHub地址。
